Path Splitter

How to configure this {% icon_help %}

Path Splitter: A delimiter to split file paths at. The part after the delimiter is kept. Useful for removing container/remote path prefixes. For example, if your logs contain /app/myproject/views.py and you set the splitter to /app, it will keep /myproject/views.py.

Path Prefix: A prefix to prepend to the split path. Used to convert container paths to your local system paths. If you set the prefix to /home/john-doe/projects, the final path becomes /home/john-doe/projects/myproject/views.py, which can then be opened in VS Code.

Example: Container path: /app/myproject/views.py → Split by /app → Prefix with /home/user/projects → Result: /home/user/projects/myproject/views.py

The settings will apply to file paths that can be opened in VS Code as well as all strings containing file paths that can be copied to the clipboard by clicking on them.
